Android App Demo

What was used ?
-------------

In this small example application, the following patterns and features have been used ... 

> **Patterns:**

> - Design Pattern M-V-P.
> - Code Conventions .
> - Material Designer.

> **Database:**
> 
> - SQL Lite.


> **External libraries:**
> 
> - HTTP client lib.
> - Animator lib.
 
> **Elements UI:**
> 
> - ConstrantLayout.
> - RecycleView.
> - CardView.
> - DatePicker.
> - CollapsedToolbar.
> - Spinner.
