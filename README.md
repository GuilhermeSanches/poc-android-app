Android App Demo
===================

----------

What was used ?
-------------

In this small example application, the following patterns and features have been used ... 

> **Patterns:**

> - Design Pattern MVP.
> - Data access object.
> - Singleton.
> - Code Conventions .
> - Material Designer.

> **Database:**
> 
> - SQL Lite.


> **External libraries:**
> 
> - HTTP client lib.
> - Animator lib.
 
> **Elements UI:**
> 
> - ConstrantLayout.
> - LinearLayout.
> - RelativeLayout.
> - RecycleView.
> - CardView.
> - DatePicker.
> - CollapsedToolbar.
> - Spinner.
> - TextView.
> - ImageView.
> - Button.
> - TextInputLayout.
> - ProgressBar.

> **Broadcast receiver:**
> 
> - ACTION_BOOT_COMPLETED.
> - ACTION_SHUTDOWN.
